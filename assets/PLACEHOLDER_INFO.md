# Placeholder Images

This directory contains placeholder images for the Krysh HVAC website. Replace these with actual professional photos of your HVAC installations and team.

## Image Requirements

### Logo (logo.svg)
- Format: SVG (scalable vector graphics)
- Dimensions: Scalable, minimum 120px wide
- Colors: Should work on both light and dark backgrounds
- Usage: Header, footer, and branding

### Favicon (favicon.ico)
- Format: ICO
- Sizes: 16x16, 32x32, 48x48 pixels
- Usage: Browser tab icon

### Hero Image (hero-hvac.jpg)
- Dimensions: 1200x600px minimum
- Subject: Professional HVAC technician at work
- Quality: High resolution, well-lit, professional
- File size: Under 200KB (optimized for web)

### Gallery Images
All gallery images should be:
- Dimensions: 400x300px minimum (4:3 aspect ratio)
- Format: JPG, optimized for web
- File size: Under 100KB each
- Subject: High-quality installation photos
- Lighting: Good natural or professional lighting

## Using Your Own Images

1. Replace the placeholder files with your actual images
2. Maintain the same file names to avoid broken links
3. Optimize images for web using tools like:
   - TinyPNG (online compression)
   - ImageOptim (Mac)
   - Squoosh.app (web-based)

4. Test all pages after replacing images to ensure they display correctly
