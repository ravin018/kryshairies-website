## Image Guidelines

### Formats
- **Photos**: JPG (optimized for web, max 150KB)
- **Graphics/Logo**: SVG (scalable, small file size)
- **Icons**: SVG or PNG (with fallbacks)
- **Favicon**: ICO format (16x16, 32x32, 48x48 sizes)

### Optimization
- Compress all images for web delivery
- Use WebP format where supported
- Implement responsive images with srcset
- Include proper alt text for accessibility

### Naming Convention
- Use descriptive, lowercase names
- Separate words with hyphens
- Include dimensions for multiple sizes
- Example: `split-system-installation-400w.jpg`

## Image Specifications

### Hero Image
- **Dimensions**: 1200x600px minimum
- **Subject**: Professional HVAC technician at work
- **Format**: JPG, optimized for web
- **Alt text**: "Professional HVAC technician installing air conditioning unit"

### Gallery Images
- **Dimensions**: 400x300px minimum
- **Subject**: High-quality installation photos
- **Format**: JPG, under 100KB each
- **Composition**: Clean, professional, good lighting

### Logo Requirements
- **Format**: SVG (scalable vector graphics)
- **Colors**: Work on white and dark backgrounds
- **Minimum size**: 120px wide
- **Clear space**: 50% of logo width on all sides
